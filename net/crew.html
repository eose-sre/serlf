> cat ENGINEERING_VALUES.md

// Ship daily

If it's not deployed, it doesn't exist. We push code every day. Small PRs, fast reviews, continuous delivery. Perfection is the enemy of shipping.

// Code is communication

Good code reads like good prose. Name things well. Comment the "why," not the "what." Future you is a stranger ‚Äî be kind to them.

// Cheap > expensive

AI tools shouldn't cost a mortgage. We price for indie devs, students, and small teams. If a product costs more than a coffee, it better be damn good.

// Automate everything

If you do it twice, script it. If you script it twice, make it a product. That's literally how half the serlf catalog was born.

// Zero meetings

We communicate in code, commits, and docs. If it can't be a comment, it can be a 2-line message. No standups. No retros. Ship or don't.

// Trust the logs

Feelings don't debug production. Logs do. Metrics do. Traces do. Every system is observable. Every failure is a learning opportunity with a stack trace.
