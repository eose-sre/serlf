API Documentation

One API. One key. Every serlf product. Rick wrote these docs himself, so they're actually accurate.

Base URL: https://api.serlf.net/v1

Authentication

All requests require a Bearer token. Get yours from the dashboard.

# Include in every request
Authorization: Bearer sk_live_your_api_key_here

# Or use the SDK
client = serlf.Client(api_key="sk_live_...")

Keys start with sk_live_ (production) or sk_test_ (sandbox). Test keys hit the same API but don't charge credits. Use them. That's why they exist.

Rate Limits

TierRequests/minRequests/dayBurst
Free201,0005/sec
Pro ($2+)12010,00020/sec
Team ($10+)600100,00050/sec

Rate limit headers are included in every response: X-RateLimit-Remaining, X-RateLimit-Reset. Read them. Respect them.

Error Handling

200OK — Request succeeded
201Created — Resource created
400Bad Request — Check your parameters
401Unauthorized — Invalid or missing API key
403Forbidden — Insufficient permissions for this resource
429Rate Limited — Slow down. Check X-RateLimit-Reset header
500Server Error — Our fault. We're on it. Check status.serlf.net
// Error response format
{
  "error": {
    "code": "rate_limited",
    "message": "Too many requests. Try again in 30s.",
    "retry_after": 30
  }
}

Code Pilot — Generate Code

POST /v1/codepilot/generate Generate code from a prompt

Parameters

NameTypeRequiredDescription
promptstringrequiredWhat you want to build
langstringTarget language (python, javascript, go, rust, etc.)
contextstringExisting code for context
stylestringconcise | verbose | documented

curl

curl -X POST https://api.serlf.net/v1/codepilot/generate \
  -H "Authorization: Bearer $SERLF_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "FastAPI endpoint for user auth with JWT",
    "lang": "python",
    "style": "documented"
  }'

Python

import serlf

client = serlf.Client(api_key="sk_live_...")
result = client.codepilot.generate(
    prompt="FastAPI endpoint for user auth with JWT",
    lang="python",
    style="documented"
)
print(result.code)

JavaScript

import { Serlf } from '@serlf/sdk';

const client = new Serlf('sk_live_...');
const result = await client.codepilot.generate({
  prompt: 'FastAPI endpoint for user auth with JWT',
  lang: 'python',
  style: 'documented'
});
console.log(result.code);

Response

{
  "id": "gen_abc123",
  "code": "from fastapi import FastAPI, Depends...",
  "lang": "python",
  "tokens_used": 847,
  "model": "codepilot-v3"
}
POST /v1/codepilot/review AI code review
NameTypeRequiredDescription
codestringrequiredCode to review
focusstringsecurity | performance | style | all
curl -X POST https://api.serlf.net/v1/codepilot/review \
  -H "Authorization: Bearer $SERLF_KEY" \
  -d '{ "code": "your code here", "focus": "security" }'
POST /v1/codepilot/refactor Refactor existing code
NameTypeRequiredDescription
codestringrequiredCode to refactor
goalstringWhat to optimize (readability, performance, etc.)
curl -X POST https://api.serlf.net/v1/codepilot/refactor \
  -H "Authorization: Bearer $SERLF_KEY" \
  -d '{ "code": "messy code", "goal": "readability" }'

Bot Factory — Create Bot

POST /v1/bots/create Create a new AI bot
NameTypeRequiredDescription
namestringrequiredBot name
platformstringrequireddiscord | slack | telegram | webhook
personalitystringSystem prompt / personality description
toolsarrayTool names the bot can use
curl -X POST https://api.serlf.net/v1/bots/create \
  -H "Authorization: Bearer $SERLF_KEY" \
  -d '{
    "name": "support-bot",
    "platform": "discord",
    "personality": "Helpful but sarcastic tech support"
  }'
POST /v1/bots/{id}/deploy Deploy bot to platform
NameTypeRequiredDescription
tokenstringrequiredPlatform bot token (Discord token, Slack token, etc.)
channelsarrayChannel IDs to activate in
GET /v1/bots/{id}/logs Get bot conversation logs
NameTypeRequiredDescription
limitintegerMax entries (default 50)
sincestringISO 8601 timestamp

Cloud Dock — Deploy

POST /v1/dock/deploy Deploy a service
NameTypeRequiredDescription
namestringrequiredService name
sourcestringrequiredGit URL or Docker image
envobjectEnvironment variables
scaleobject{ min: 1, max: 5 }
curl -X POST https://api.serlf.net/v1/dock/deploy \
  -H "Authorization: Bearer $SERLF_KEY" \
  -d '{
    "name": "my-api",
    "source": "github.com/you/repo",
    "env": { "NODE_ENV": "production" }
  }'
GET /v1/dock/services List deployed services 
curl https://api.serlf.net/v1/dock/services \
  -H "Authorization: Bearer $SERLF_KEY"

Data Vault — Store

POST /v1/vault/store Store encrypted data
NameTypeRequiredDescription
keystringrequiredStorage key
valueanyrequiredData to store (JSON, string, binary)
ttlintegerTime-to-live in seconds
curl -X POST https://api.serlf.net/v1/vault/store \
  -H "Authorization: Bearer $SERLF_KEY" \
  -d '{
    "key": "user:preferences",
    "value": { "theme": "dark" },
    "ttl": 86400
  }'
GET /v1/vault/{key} Retrieve stored data 
curl https://api.serlf.net/v1/vault/user:preferences \
  -H "Authorization: Bearer $SERLF_KEY"
